1. Bij het inladen van een bestand is het noodzakelijk om de locatie van het
bestand aan te geven. Dit kan op 2 manieren:
● Absolute path
● Relative path
Wat houden deze 2 paths in?

absolute: wijst target aan vanaf de root, adres verandert niet

relative: wijst target aan ten opzichte van een zekere locatie, bijv. de current working directory, target-adres verandert afhankelijk van wat de huidige, actieve locatie is

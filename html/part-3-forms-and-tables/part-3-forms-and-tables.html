HTML tables

10. Om een tabel te maken heb je volgende elementen nodig:
● table
● thead
● tbody
● tfoot
● th
● tr
● td
Geef voor elke element een definitie.

table: definieert een HTML-tabel

thead: groepeert de headers van een tabel

tbody: groepeert de bulk van de tabeldata

tfoot: groepeert de totalen of opsommingen van een tabel

th: definieert een tabelheader

tr: definieert een rij in een tabel

td: definieert de data van één cel

push(): add an element at the end of an array

pop(): remove an element at the end of an array

shift(): remove an element at the beginning of an array

unshift(): add an element at the beginning of an array

slice(): copies part of an existing array to create a new array. The source array is left unchanged after the slicing.

splice(): add new items to an array. The splice method has two or more parameters: 1. index number where items should be added 2. number of items to be deleted 3. any items to be added to the array

sort(): sort an array alphabetically. Not to be used with number arrays, for it will produce faulty results.

reverse(): reverse the order of an array.

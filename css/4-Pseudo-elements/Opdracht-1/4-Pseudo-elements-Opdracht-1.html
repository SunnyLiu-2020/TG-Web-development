4-Pseudo-elements-Opdracht-1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

  1. ::before
  2. ::after
  3. ::first-line
  4. ::first-letter

Before: inserts content before an HTML element.

After: inserts content after an element.

First-line: applies certain style to the first line of an HTML element.

First-letter: applies certain style to the first letter of an element.

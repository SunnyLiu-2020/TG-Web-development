6-Transitions-Opdracht-1

Zoek uit wat transitions zijn en beschrijf met eigen woorden wat het betekent.

Transition is a functionality in CSS to control the animation speed of changes in CSS properties. It's not meaningful to change all properties, so the list of animatable properties is finite but it changes over time as the specification develops.

The transition property is shorthand for four properties:

  1. transition-property
  2. transition-delay
  3. transition-duration
  4. transition-timing-function

Transition-property: here you can explicitly state what CSS property or properties you want to animate.

Transition-delay: this is where you can define how long you want to delay the start of an animation.

Transition-duration: this is where you tell the browser how long the animation should last.

Transition-timing-function: here you can specify how the animation should take place, for example linearly or quick at the beginning and slow at the end.

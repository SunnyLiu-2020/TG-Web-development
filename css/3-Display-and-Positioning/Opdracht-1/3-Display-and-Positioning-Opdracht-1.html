3-Display-and-Positioning-Opdracht-1

Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

There are five position values:

  1. fixed
  2. absolute
  3. relative
  4. static
  5. sticky

Fixed: the position of an HTML element is set relative to the viewport.

Absolute: the position of an HTML element is defined relative to the closest positioned ancestor. If thhere's no ancestor, then the element is positioned relative to the container it is in.

Relative: the position of an HTML element is relative to its normal position.

Static: the position of an HTML element is in its usual place. This is the default position value for HTML elements.

Sticky: the position of an HTML element is relative to its original position, but when the outer reaches of a container or viewport have been reached by way of scrolling the element will be fixed. So sticky position is a combination of relative and fixed position.

Er zijn verschillende selectors, bijvoorbeeld;
• type selectors
• descendant selectors
• class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Type: bedoeld wordt de verschillende soorten HTML-elementen, bijv. h1, h4 of p. Een voordeel van type selectors is dat je heel snel alle gevallen van een bepaald type HTML-element binnen een webpagina vorm kunt geven.

Class: Deze selectors richten zich op elementen die zijn gedefinieerd als behorend tot een zekere class. Het voordeel van deze selector is dat je heel makkelijk specifieke elementen binnen div-tags kunt kiezen.

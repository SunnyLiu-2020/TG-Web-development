1-Selectors-and-Visual-rules-Opdracht-1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. external
  2. internal
  3. inline

Ad 1

External: er is een apart CSS-document dat de stijlen specificeert. In <head> staat in <link> de verwijzing naar het desbetreffende document.

<link rel="stylesheet" href="">

Ad 2

Internal: In <head> staan in <style> de CSS-stijlen van die webpagina.

Ad 3

Inline: CSS wordt direct toegepast op een HTML-element binnen de tag van dat element.

De cascading-prioriteit is:

  1. inline
  2. external/internal
  3. browser default

Als inline, internal en external CSS met elkaar concurreren om stijlen toe te passen op hetzelfde element, zal inline dus domineren.

De prioriteit tussen external en internal CSS-stijlen wordt bepaald door welke als laatste in het head-element staat.

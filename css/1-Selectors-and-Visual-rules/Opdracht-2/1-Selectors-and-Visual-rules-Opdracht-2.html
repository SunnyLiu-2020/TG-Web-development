CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

CSS-syntax bestaat uit drie delen: selector, property en value. Een selector heeft een of meerdere HTML-elementen als onderwerp. Daarover doet CSS een uitspraak (Engels: declaration) die bestaat uit een eigenschap en waarde (property and value). Eigenschap en waarde worden gescheiden door een dubbele punt. Na de waarde volgt een puntkomma. De declaratie staat tussen accolades. Voorbeeld:

p {text-align: center;}

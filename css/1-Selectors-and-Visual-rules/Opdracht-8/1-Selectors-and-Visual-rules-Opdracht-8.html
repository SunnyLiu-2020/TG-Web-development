Cascade

Cascading Style Sheets means that styles are defined in different ways for HTML elements, for example inline, internal or external. That is what Style Sheets refer to. There can be more than one sheet, because there are different ways to apply the styles. The word Cascading refers to the application process of these styles, which takes place consecutively like a cascade, a waterfall that flows down one step at a time like the individual steps of a staircase. The priority of the cascading process is:

  1. inline
  2. external / internal, depending on whether <link> or <style> comes last in <head>. The one last-mentioned in the head tag takes precedence, that is it will win and dictate its style if the external and internal styles compete. This is because in the cascading process the HTML document is read from top to bottom.
  3. browser default

The background color of inline.html is blue, because inline styles have priority over the competing internal and external styles that define different background colors.

The background color of internal.html is green, because the style tag for internal CSS styling is mentioned last in the HTML document as compared to the link tag, which links to an external CSS style document and is mentioned earlier. Please note that the inline style declaration in the body tag has been deleted.

The background color of external.html is pink, because the link tag for external CSS styling is mentioned last in the HTML document as compared to the style tag. Since the external style is read last by the browser, its definition of a pink background color has precedence over the green color of the internal style.

Inheritance

The inheriting of style properties from its parent element. If the parent doesn't have any particular styles, CSS will continue to look for the style properties of the parent's parent element all the way until a style property is found. And then the style will be applied to the child or descendant element, which has a style property set, not to aa value, but to the CSS keyword inherit.

  • inherit

CSS defines the span element as being blue. However, there's an extra style declaration for span elements that are nested within an element with class="extra". The article from the MDN on cascading and inheritance teaches us that class selectors have a value of 10 and type / name / element selectors have a value of 1. Since the class selector has a higher value than the element selector, the span element in the second div will inherit the green color from its parent.
